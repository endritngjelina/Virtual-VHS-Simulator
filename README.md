# Virtual VHS Simulator

A retro web experience that brings back the nostalgic feel of VHS tapes through your browser. Each simulation recreates the distinctive visual artifacts, tracking issues, and warm analog aesthetic of vintage video cassettes.

## Features

- Multiple VHS tape simulations (13+ unique experiences)
- Authentic VHS visual effects including:
  - Scan lines and tracking noise
  - Color bleeding and distortion
  - Chromatic aberration
  - Tape wear and aging effects
- Pure HTML/CSS/JavaScript implementation
- No framework dependencies

## Usage

Simply open any of the HTML files in your web browser:
- `Virtual VHS Simulator 0.html` through `Virtual VHS Simulator 12.html`

Each file represents a different "tape" with unique content and effects.
